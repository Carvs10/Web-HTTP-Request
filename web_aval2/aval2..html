Atividade da Unidade 2!

Sua aplicação deve utilizar o arquivo com a função AJAX fornecida:

  • Para popular o SELECT com a lista de usuários;
  • Obter a lista de TODOS de um usuário selecionado

Ao selecionar um usuário, envie como 'param' o valor do atributo "id" dele, como reposta você terá um JSON com a lista de 'TODOS' do susário selecionado.

Em seguida ger um gráfico de Pizza indicando as tarefas realizadas e as não realizadas. O atributo 'completed' presente no JSON é o dado a ser contabilizado para os gráficos.
